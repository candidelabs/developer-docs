### simulateUserOperationWithTenderlyAndCreateShareLink
Simulates a UserOperation via the EntryPoint's handleOps on Tenderly and returns a shareable dashboard link.

**Parameters:** `tenderlyAccountSlug`, `tenderlyProjectSlug`, `tenderlyAccessKey`, `chainId`, `entrypointAddress`, `userOperation` (V6/V7/V8/V9), `blockNumber` (optional), `stateOverrides` (optional).

#### Source code
[simulateUserOperationWithTenderlyAndCreateShareLink](https://github.com/candidelabs/abstractionkit/blob/main/src/utilsTenderly.ts#L91)

### simulateUserOperationCallDataWithTenderlyAndCreateShareLink
Simulates UserOperation callData with Tenderly and returns shareable links. Unlike `simulateUserOperationWithTenderlyAndCreateShareLink`, this simulates the inner call data rather than the full EntryPoint handleOps flow.

#### Source code
[simulateUserOperationCallDataWithTenderlyAndCreateShareLink](https://github.com/candidelabs/abstractionkit/blob/main/src/utilsTenderly.ts#L403)

### simulateSenderCallDataWithTenderlyAndCreateShareLink
Simulates sender callData directly with Tenderly and returns shareable links. Useful for simulating the execution of callData from a specific sender address without wrapping it in a full UserOperation.

#### Source code
[simulateSenderCallDataWithTenderlyAndCreateShareLink](https://github.com/candidelabs/abstractionkit/blob/main/src/utilsTenderly.ts#L619)

## EIP-7702 Utilities

### createAndSignEip7702DelegationAuthorization
Creates and signs an EIP-7702 delegation authorization, allowing an EOA to delegate its code to a specified contract address. Accepts either a hex-encoded private key string or an async signer callback `(hash: string) => Promise<string>` for use with viem, ethers Signers, hardware wallets, or MPC signers.

#### Source code
[createAndSignEip7702DelegationAuthorization](https://github.com/candidelabs/abstractionkit/blob/main/src/utils7702.ts#L131)

### createRevokeDelegationAuthorization
Creates a signed authorization that revokes an existing EIP-7702 delegation by setting the delegatee address to the zero address, restoring the EOA to a normal account.

#### Source code
[createRevokeDelegationAuthorization](https://github.com/candidelabs/abstractionkit/blob/main/src/utils7702.ts#L187)

### createEip7702DelegationAuthorizationHash
Computes the keccak256 hash of an EIP-7702 delegation authorization using the MAGIC prefix (0x05) as defined in the EIP-7702 spec. Useful when signing the authorization externally.

#### Source code
[createEip7702DelegationAuthorizationHash](https://github.com/candidelabs/abstractionkit/blob/main/src/utils7702.ts#L206)

### createAndSignEip7702RawTransaction
Creates and signs a raw EIP-7702 (set-code, type 0x04) transaction with an authorization list. The transaction is RLP-encoded and includes the type prefix.

#### Source code
[createAndSignEip7702RawTransaction](https://github.com/candidelabs/abstractionkit/blob/main/src/utils7702.ts#L258)

### createEip7702TransactionHash
Computes the keccak256 hash of an EIP-7702 transaction for signing purposes, using the type 0x04 prefix and RLP encoding.

#### Source code
[createEip7702TransactionHash](https://github.com/candidelabs/abstractionkit/blob/main/src/utils7702.ts#L323)
